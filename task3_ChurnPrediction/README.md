#  Task 3: Customer Churn Prediction

##  Overview

This project predicts whether a customer will **churn or stay** using a **Logistic Regression model** in Python.

##  Tech Stack

Python, Pandas, NumPy, Scikit-learn

##  How to Run

1. Install dependencies:
   pip install pandas numpy scikit-learn

2. Place `churn.csv` in the project folder

3. Run:
   python task3.py

##  Features

* Data preprocessing
* Model training
* Accuracy & classification report
* User input prediction

##  Note

User input is not fully used in prediction and can be improved.

##  Output

Predicts: **Customer will CHURN** or **STAY**
