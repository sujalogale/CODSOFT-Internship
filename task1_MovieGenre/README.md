# Task 1: Movie Genre Prediction

##  Description
This project predicts the genre of a movie based on given features like plot, title, or other attributes using machine learning.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

##  Dataset
The dataset contains information about movies such as:
- Movie title
- Description / plot
- Genre labels

##  How It Works
- Load dataset using pandas
- Clean and preprocess text data
- Convert text into numerical format (like TF-IDF)
- Train machine learning model
- Predict movie genre

##  Output
The model predicts the genre of the movie (like Action, Comedy, Drama, etc.) with accuracy.
